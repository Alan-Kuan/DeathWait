# DeathWait死亡等待

## About

This is a plugin about player respawn system for Minecraft Spigot server.

Author: 根號N (Alan Kuan)

Latest Version: 4.0

## Features

* Respawn countdown
* Custom respawn points
* Custom respawn items
* GUI for selecting respawn points
* SOS system
* And more...
